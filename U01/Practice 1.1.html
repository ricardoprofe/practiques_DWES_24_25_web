1 Practice 1.1. Setup of the PHP programming environment

You don’t need to submit this practice, but doing it is necessary in order to follow the course and to do the rest of the practices

1.1 Create a local web server

Following the instructions of Unit 1, you have to create a local web server for development. You can install XAMPP on the operating system of your choice, or you can install a Linux server as explained in the Unit.

You can choose to do the installation in a Virtual Machine, but it is recommended to install it in a real machine.

1.2 Install and configure a code editor

Install Visual Studio Code on your programming environment. Install the PHP Tools extension too.

Additionally, you can install PhpStorm.
